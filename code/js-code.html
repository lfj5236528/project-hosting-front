样例题目及程序

A + B
描述

给予两个整数 a 和 b ，计算它们的和 a + b

输入

输入包含多组测试数据，每一行包含两个整数 a 和 b

输出

在一行中输出 a + b 的值

样例输入
1 2
3 4
5 6
样例输出
3
7
11
JavaScript 语言
var line;
while(line = read_line()){
    line = line.split(' ');
    print(parseInt(line[0]) + parseInt(line[1]));
}
系统使用说明
注意事项:

1.由于js编程练习算法题,需要安装V8引擎来模拟数据输入的过程,所以利用本系统可以不用安装V8引擎,进行模拟数据输入过程。

2.由于使用javascript进行算法编程时,本地环境是无法进行过程调试的，所以我们可以借助本系统配合浏览器进行过程调试。

3.使用本系统,我们首先在数据填充区输入我们要的测试用例的行数,并点击输入数据按钮,接着需要输入每行的数据，数据间的分隔符由题目确定，最后在代码填充区填入编写的javascript代码，点击运行，即可得到测试结果。

4.使用调试代码请打开开发者工具。

4.读入每行数据请使用read_line()或readline()函数。
